EFLAGS register

EFLAGSFlags. The state of various conditions in the cpu. The purpose of individual bits of the EFLAGS register is described below.

15141312 11100908 07060504 03020100
  Nested Task I/O privilege level Overflow Direction Interrupt Trap Sign Zero   Auxiliary Carry   Parity   Carry
  • Carry (CF): Holds the carry out after addition or the borrow after subtraction. Also indicates error conditions.
  • Parity (PF): 0 for odd number of bits and 1 for even. Obsolete feature of the 80x86.
  • Auxiliary Carry (AF): Highly specialized flag used by DAA and DAS instructions after BCD addition or subtraction.
  • Zero (ZF): 1 if the result of an arithmetic or logic instruction is 0.
  • Sign (SF): 1 if the sign of the result of an arithmetic or logic instruction is negative.
  • Trap (TF): Trap enable. The microprocessor interrupts the flow of instructions on conditions indicated by the debug and control registers.
  • Interrupt (IF): Controls the operation of the INTR (Interrupt request) pin. If 1, interrupts are enabled. Set by STI and CLI instructions.
  • Direction (DF): Specifies increment or decrement mode for the DI and/or SI registers during string instructions. If 1, registers are automatically decremented. Set by STD and CLD instructions.
  • Overflow (OF): Set for addition and subtraction instructions.
  • I/O privilege level (IOPL): Holds the privilege level at which your code must be running in order to execute any I/O-related instructions. 00 is the highest.
  • Nested Task (NT): Set when one system task has invoked another through a CALL instruction in protected mode.

  • Resume (RF): Used with debugging to selectively mask some exceptions.
  • Virtual Mode (VM): When 0, the CPU can operate in Protected mode, 286 Emulation mode or Real mode. When set, the CPU is converted to a high speed 8086. This bit has enormous impact.
  • Alignment Check (AC): Specialized instruction for the 80486SX.
  • Virtual Interrupt Flag (VIF): Copy of the interrupt flag bit.
  • Virtual Interrupt Pending (VIP): Provides information about a virtual mode interrupt.
  • Identification (ID): Supports the CPUID instruction, which provides version number and manufacturer information about the microprocessor.
